# 実装する必要のある機能一覧

## 1. MVPで必須
### 1.1 タイマー基本機能
1. 作業時間カウントダウン開始
2. 一時停止と再開
3. リセット
4. セッション完了判定
5. 残り時間表示（mm:ss）

### 1.2 タイマー状態管理
1. 待機中・作業中・一時停止の状態保持
2. 状態遷移ルール（無効操作の抑止）
3. タブ復帰時も正しい残り時間を維持

### 1.3 進捗表示UI
1. 円形プログレスリング
2. 現在モード表示（例: 作業中）
3. 開始・リセットボタン
4. 今日の進捗カード（完了数・集中時間）

### 1.4 今日の統計集計
1. 完了ポモドーロ数の加算
2. 集中時間の累積
3. 日付単位での表示

### 1.5 永続化（最低限）
1. ブラウザ再読み込み後の状態復元
2. 今日の進捗の保持

## 2. MVP後に追加推奨
### 2.1 休憩機能
1. 短休憩への自動遷移
2. 長休憩（一定回数ごと）
3. 作業・休憩サイクル管理

### 2.2 設定機能
1. 作業分数の変更
2. 短休憩・長休憩分数の変更
3. 長休憩に入る間隔の変更

### 2.3 通知機能
1. セッション完了時の音通知
2. ブラウザ通知
3. タイトルやバッジでの状態通知

### 2.4 バックエンド連携
1. 統計取得API
2. セッション記録API
3. 設定取得・更新API

### 2.5 履歴・可視化
1. 日別の実績一覧
2. 週次集計
3. 完了率や連続達成日数

## 3. 品質面で必要
### 3.1 テスト
1. タイマー状態遷移の単体テスト
2. 時間計算の境界値テスト
3. APIの入力検証テスト

### 3.2 UX・アクセシビリティ
1. モバイル対応
2. キーボード操作対応
3. コントラストと読みやすさの確保

### 3.3 信頼性
1. 異常時のエラーメッセージ
2. データ破損時の復旧方針
3. ログ出力の最小整備
